Project Name: Elizabeth's Bakery

Overview

The goal of this project is to design a website for a baker. The site will showcase a variety of desserts and cake styles, allowing visitors to explore different options. It will also provide pricing information and the most effective ways to contact the baker.

Wireframe

Shared Components

  • JavaScript file for interactivity (script.js)

Pages

index.html ~ Home Page
  • Introduction and welcome message
  • Navigation to other pages
menu.html ~ Menu Page
  • List of dessert and pastry varieties
  • Navigation to return to the home page
gallery.html ~ Gallery Page
  • Visual display of the best desserts
  • Images may include descriptions
  • Navigation to other pages
form.html ~ Order Form Page
  • Input fields for custom orders
  • Data validation
  • Submission triggers an email notification to the site admin
  • Navigation to other pages
faq.html ~ FAQs Page
  • Common questions about prices and ordering
  • Helps reduce confusion
  • Navigation to other pages

Dynamic Functionality

  • A form will be provided for customers to submit orders.
  • Upon submission, the admin will receive an email notification.
